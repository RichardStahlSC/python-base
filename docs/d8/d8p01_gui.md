# Demo: Interface gráfica

Aplicações para desktop utilizam o sistema de janelas (window manager) ativo no sistema operacional para posicionar e renderizar os elementos.

Geralmente essas aplicações são criadas utilizando toolkits, que são o conjunto de `bindings` para a criação
de janelas, botões etc...

Principais toolkits:

- Tkinter
- PyQt/Pyside
- PyGTK/Glade
- WxPython
- Kivy

## Tkinter

O Tkinter é um dos principais toolkits desktop para Python
a sua principal caracteristica é ser leve e portável.

Ele é baseado na linguagem Tcl/Tk e seus bindings já vem embutido no Python [https://docs.python.org/3.10/library/tkinter.html](https://docs.python.org/3.10/library/tkinter.html) podendo ser necessário instalar Tcl/Tk no sistema operacional.

Teste se o tk está disponível em sua instalação do Python.

```bash
python -m tkinter
```
